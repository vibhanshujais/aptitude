Insertion Sort

What is Insertion Sort?

Insertion sort is a simple sorting algorithm that builds the final sorted array one element at a time. It is much less efficient on large lists than more advanced algorithms such as quicksort, heapsort, or merge sort, but it performs well on small lists or partially sorted lists.

How Insertion Sort works?

  1. Initialization:
    • The first element in the array is considered to be sorted.
    • Start from the second element (index 1) and consider it as the key.
  2. Comparison and Insertion:
    • Compare the key with the element to its left.
    • If the key is smaller, swap it with the element to its left.
    • Repeat this process until the key is greater than or equal to the element to its left or until it reaches the beginning of the array.
  3. Move to the Next element:
    • Move to the next element in the array (increment the index).
    • Repeat steps 2 and 3 until the entire array is sorted.

Example:

Consider initial array : [7, 4, 5, 2, 1]

  • Pass 1: Swap 7 and 4. Array becomes [4, 7, 5, 2, 1].
  • Pass 2: Swap 7 and 5. Array becomes [4, 5, 7, 2, 1].
  • Pass 3: Insert 2 in its correct position. Array becomes [2, 4, 5, 7, 1].
  • Pass 4: Insert 1 in its correct position. Array becomes [1, 2, 4, 5, 7].

Complexity Analysis:

Time Complexity:

  • Best Case:O(N)
  • Average Case:O(N2)
  • Worst Case:O(N2)

Space Complexity:

  • O(1)

Advantages:

  • Simple Implementation
  • Efficient for Small Data Sets
  • In-Place Sorting

Disadvantages:

  • Not a Stable Sort
  • Inefficiency for Large Data Sets
